# SPIDERMAN

> Minimalist distributed web crawler for JavaScript

## Installation

### Requirements

- Node.js >= 8.0.0

### Instructions

- With npm:

```bash
npm i @albert-team/spiderman@canary
```

- With yarn:

```bash
yarn add @albert-team/spiderman@canary
```
