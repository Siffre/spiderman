# Spiderman

Minimalistic web crawler for Node.js

## INSTALLATION

### Requirements

- Node.js >= 14

### Instructions

- With npm:

```sh
$ npm i @albert-team/spiderman
```

- With yarn:

```sh
$ yarn add @albert-team/spiderman
```
